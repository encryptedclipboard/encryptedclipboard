## 🚀 The Product

**Encrypted Clipboard Manager** is a privacy-first, local-first application designed for developers, security professionals, and anyone who values their data's integrity. It securely manages your clipboard history and synchronizes it across all your devices using state-of-the-art encryption.

### Peak Privacy with Zero-Knowledge
Unlike traditional clipboard managers, our tool ensures your data never touches a server in an unencrypted state. Your master password is the only key, and it never leaves your device.

---

## 🔥 Key Features

- 🔐 **Zero-Knowledge Encryption**: AES-GCM encryption handled entirely on the client-side.
- 🔄 **Secure Cloud Sync**: Seamless synchronization across devices without compromising privacy.
- 💻 **Local-First Architecture**: Blazing fast performance, works completely offline.
- 🛠️ **Developer Friendly**: Built by developers, for developers. Clean UI, keyboard shortcuts, and power-user features.
- 📦 **Cross-Platform**: Available as a browser extension and progressive web app.

## 🛠️ The Tech Behind the Privacy

We pride ourselves on using a modern, efficient, and secure tech stack:

- **Extension**: [Svelte.js](https://svelte.dev)
- **Frontend**: [Vue.js](https://vuejs.org/) & [Nuxt 3](https://nuxt.com/) for a reactive and performant UI.
- **Backend**: [Express](https://expressjs.com/) & [Bun](https://bun.sh/) for a high-performance, secure API.
- **Database**: [MongoDB](https://www.mongodb.com/) with [Mongoose](https://mongoosejs.com/) for reliable data persistence.
- **Encryption**: Robust implementation based on the Web Crypto API.
- **Package Management**: Powered by [Bun](https://bun.sh/) for lightning-fast builds.
